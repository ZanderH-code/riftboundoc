# riftboundoc

Live site: https://zanderh-code.github.io/riftboundoc/

## 项目简介

`riftboundoc` 是一个 Riftbound 资料站，聚合官方 FAQ、Errata、Rules 与卡牌信息，提供统一、可检索的阅读入口，方便快速查找最新规则与裁定。

## 功能

- FAQ 列表与详情页
- Errata 列表与详情页
- Rules 页面（含文本化规则内容）
- Cards 卡牌图库与筛选
- 卡牌详情关联内容（Related FAQ / Errata / Rules）
- 全站搜索（跨 FAQ、Errata、Rules、Cards）
- Updates 最近更新列表
